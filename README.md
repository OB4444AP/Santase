# Santase (Sixty-Six) - Course Project 02

A C++ console-based implementation of the popular card game **Santase** (also known as *Sixty-Six*). This project was developed as a course assignment for "Introduction to Programming" at the Faculty of Mathematics and Informatics (FMI), Sofia University.

## ðŸŽ® Game Description
Santase is a strategy card game for two players played with a deck of 24 cards (Ace, 10, King, Queen, Jack, 9). The goal is to be the first to reach **66 points** in a round to win game points. The first player to reach **11 game points** wins the entire match. You can type "rules" for explanation in detail.

### Key Features
* **Full Game Loop:** Play complete matches with automatic shuffling, dealing, and scoring.
* **Special Moves:** Implements "Marriage" (20/40 points), "Switch Nine" (swapping the 9 for the trump), and closing the talon.
* **Save/Load System:** Save your game progress to a text file and resume later.
* **Settings:** Customizable rules (points to win, marriage bonuses, last trick bonus).
